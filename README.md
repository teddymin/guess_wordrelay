A sample command-line application with an entrypoint in `bin/`, library code
in `lib/`, and example unit test in `test/`.
### 프로젝트의 목적
Dart 학습 및 프로그램 제작
### 주요 기능    
간단한 끝말잇기   
### 특징   
두음법칙X    
### 빌드/사용방법    
터미널에서 dart run 입력   
닉네임 입력 이후    
끝말잇기 시작
### 사전 요구사항
표준국어대사전 오픈 API 서비스 인증 키   
관련 링크:https://stdict.korean.go.kr/openapi/openApiInfo.do   
이후 코드 14번 줄 
```
String api_key = 'YOUR_API_KEY';
```
YOUR_API_KEY를 API 서비스 인증키로 수정
